# Proyecto Integrador - Jardinería
Este repositorio contiene el desarrollo completo de un proceso ETL aplicado a la BD Jardinería.

## Contenido
- Modelo estrella
- Implementación de base de datos staging
- Scripts SQL de creación y carga de datos
- Pruebas de calidad de datos
- Backup de la BD
